    Принимаем:
- Одеяла, ковры, постельное бельё - передаём зооволонтёрам;
- Флаконы с дозаторами. Тара от средств: для чистки стёкол, мытья посуды, пола - сдаём в Чистую пользу для повторного использования;
- Сломанные зонты - передаём мастерам для пошива бахил;
- Батарейки - сдаём в пункт сбора батареек;
